# NUDIS-T5.1 — Lexique couleur, texture et motif
*Vocabulaire taxonomique de référence pour la description des opisthobranchs*
*Source principale : Gosliner T. et al., Nudibranch & Sea Slug ID — Indo-Pacific*

---

## 1. COULEURS

### Principes généraux

Les descriptions taxonomiques utilisent des **termes latins ou latinisés** plutôt que des noms courants. La couleur est toujours qualifiée : intensité, distribution, nuance.

**Suffixes modificateurs :**
| Suffixe | Signification | Exemple |
|---------|--------------|---------|
| -âtre / -ish | tirant vers | blanchâtre, rougeâtre, brunâtre |
| -acé / -aceous | de la couleur de | ochracé (ocre), violacé, rosacé |
| pâle | clair, délavé | jaune pâle |
| vif / intense | saturé | orange vif |
| sombre / foncé | valeur basse | brun sombre |
| irisé / nacré | reflets | blanc nacré, irisé |

## 2. TRANSLUCIDITÉ

| Terme | Définition | Exemple chez les opisthobranchs |
|-------|-----------|--------------------------------|
| **Opaque** (*opaque*) | Imperméable à la lumière — la coloration interne n'est pas visible | Phyllidia sp., Hexabranchus |
| **Translucide** (*translucent*) | Laisse passer la lumière sans être transparent — on devine les organes internes | Aeolidiella sp., Flabellina cérates |
| **Semi-transparent** (*semi-transparent*) | Intermédiaire, partiellement transparent | Tergipes sp. |
| **Hyalin** (*hyaline*) | Transparent comme le verre, incolore | Extrémités des cérates de Cuthona sp. |
| **Diaphane** (*diaphanous*) | Très fin et transparent | Ailes natatoires de Gastropteridae |
| **Irisé** (*iridescent*) | Avec des reflets changeants selon l'angle | Glaucus atlanticus |
| **Nacré** (*pearlescent*) | Reflets nacrés doux | Chromodoris sp. fond |

## 3. TEXTURES

### Terminologie de surface
| Terme | Définition précise | Différence clé |
|-------|--------------------|---------------|
| **Lisse** (*smooth / glabrous*) | Surface sans relief, plane au toucher | Référence zéro — tout ce qui n'est pas lisse est qualifié |
| **Granuleux** (*granulate*) | Parsemé de granules sphériques fins, réguliers | Granules < tubercules ; réguliers et denses |
| **Papilleux** (*papillose*) | Couvert de papilles — projections courtes, molles, digitées | Molles (≠ tubercules rigides) |
| **Tuberculeux / tuberculé** (*tuberculate*) | Portant des tubercules — bosses fermes, bien définies, souvent bicolores | Rigides, individualisés (ex. Phyllidiidae) |
| **Rugueux** (*rugose*) | Surface irrégulière avec rides ou aspérités | Aspérités non individualisées (≠ tubercules) |
| **Ridé** (*wrinkled / plicate*) | Plissé, avec des plis irréguliers | Plis souples |
| **Spiculeux** (*spiculate*) | Portant des spicules (aiguilles calcaires ou siliceuses) | Caractéristique des éponges hôtes absorbées |
| **Velouté** (*velvety*) | Aspect mat doux, comme du velours | Impression optique, pas de relief |
| **Glanduleux** (*glandular*) | Portant des glandes visibles ou sécrétrices | |
| **Coriace** (*coriaceous*) | Épais et ferme comme du cuir | Texture générale du corps |
| **Membraneux** (*membranous*) | Fin, flexible, translucide | Voiles, parapodes |
| **Charnu** (*fleshy*) | Épais, mou, charnu | Corps de Pleurobranchus |

### Termes spécifiques aux appendices
| Terme | Définition |
|-------|-----------|
| **Pinnulé** (*pinnate*) | Avec des pinnules (branches latérales) — branchies |
| **Bipennaté** (*bipinnate*) | Pinnules portant elles-mêmes des pinnules |
| **Lamellé** (*lamellate*) | Avec des lamelles empilées — rhinophores |
| **Annelé** (*annulate*) | Avec des anneaux — rhinophores de certains doridacés |
| **Claviforme** (*clavate*) | En forme de massue — rhinophores élargis à l'apex |
| **Perfolié** (*perfoliate*) | Lamelles entourant complètement le rhinophore |
| **Digitiforme** (*digitate*) | En forme de doigt |

## 4. MOTIFS (Patterns)

### Motifs structurés
| Terme | Définition | Exemple visuel |
|-------|-----------|---------------|
| **Ponctué** (*punctate*) | Parsemé de points discrets | Taches rondes fines sur Glossodoris |
| **Maculé** (*maculate*) | Portant des macules — taches irrégulières plus grandes | Chromodoris sp. taches irrégulières |
| **Ocellé** (*ocellate*) | Avec des ocelles — taches en forme d'œil (anneau + centre) | Halgerda batangas |
| **Réticulé** (*reticulate*) | Réseau de lignes formant un treillis (filet) | Notum de Discodoris sp., Gymnodoris |
| **Vermiculé** (*vermiculate*) | Lignes sinueuses irrégulières ressemblant à des vermicelles | Platydoris sp. |
| **Strié** (*striate*) | Avec des stries — lignes fines parallèles | Rhinophores striés |
| **Fascié** (*fasciate*) | Avec des bandes larges (fascia) | Bandes transversales |
| **Rayé** (*striped*) | Lignes longitudinales continues | Chromodoris magnifica |
| **Annelé** (*annulate*) | Avec des anneaux | Cérates annelées de Flabellina |
| **Linéolé** (*lineolate*) | Avec de fines lignes discontinues | |
| **Réticulé marginal** | Réseau localisé au bord du manteau | Chromodoris sp. |

### Distribution spatiale
| Terme | Signification |
|-------|--------------|
| **Médian / médiane** | Au centre du notum / corps |
| **Marginal / marginé** | Au bord du manteau |
| **Submarginale** | Juste en dedans du bord |
| **Basal / basale** | À la base d'un appendice |
| **Subapical** | Sous l'apex (≈ sommet) |
| **Distal** | Éloigné du corps / de l'axe |
| **Proximal** | Proche du corps / de l'axe |
| **Dorsal** | Face supérieure |
| **Ventral** | Face inférieure |
| **Latéral** | Sur les flancs |
| **Antérieur** | Vers la tête |
| **Postérieur** | Vers la queue |

## 5. STRUCTURE D'UNE DESCRIPTION TAXONOMIQUE TYPE

Ordre canonique utilisé dans Gosliner et al. :

```
1. Dimensions (longueur vivant en mm)
2. Forme générale du corps
3. Coloration générale du notum/manteau
4. Détail des marges du manteau
5. Rhinophores : couleur, texture, forme
6. Branchies : nombre de pinnules, couleur
7. Pied et zone ventrale
8. Oral veil / tentacules oraux
9. Variations individuelles observées
```

**Exemple de description bien structurée** *(d'après Gosliner style)* :

> « Spécimen de 32 mm. Corps allongé, légèrement contracté. Notum blanc laiteux opaque, parsemé de granules irréguliers blancs. Marge du manteau portant une ligne submarginale orange vif, continue. Rhinophores translucides, annelés d'orange à leur base, apex blanc ; lamelles perfoliées au nombre de 18. Branchie composée de 6 pinnules bipennées, translucides avec axe blanc. Pied blanc crème, lisse. »

## 7. PIÈGES FRÉQUENTS

| Erreur | Correct |
|--------|---------|
| "Blanc avec des taches oranges" | "Blanc laiteux opaque, portant des macules orange vif distribuées irrégulièrement sur le notum" |
| "Antennes jaunes" | "Rhinophores jaune ochracé, claviformes, avec 12 lamelles perfoliées" |
| "Texture bosselée" | "Notum tuberculé, tubercules fermes, arrondis, de taille homogène, blancs à apex brun" |
